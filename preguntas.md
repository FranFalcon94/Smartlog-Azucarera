# SmartLog Azucarera 🏭🍬

Sistema de monitorización industrial basado en sensores simulados para una fábrica azucarera, orientado a la integración de entornos IT y OT.

## 📌 Descripción
SmartLog Azucarera recoge datos de sensores industriales simulados (temperatura, humedad y estado de maquinaria), los almacena de forma local y realiza copias de seguridad en la nube, permitiendo trazabilidad y análisis histórico.

El proyecto está diseñado como un sistema ligero y educativo, aplicando conceptos de:
- Ciclo de vida del dato
- Seguridad
- Cloud Computing
- Tecnologías Habilitadoras Digitales (THD)

## ⚙️ Tecnologías utilizadas
- Python 3
- SQLite
- JSON
- Simulación de IoT
- GitHub (Open Source)
